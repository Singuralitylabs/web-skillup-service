# GAS学習

Google Apps Script を使った自動化と開発の基礎を学びます。

## フェーズ一覧

| フェーズ | タイトル | 週数 | 概要 |
|---------|---------|------|------|
| [Phase 1](./phase1.md) | 基礎文法 | 2週 | JavaScriptの基礎文法を学びます |
| [Phase 2](./phase2.md) | Googleドライブ | 1週 | Googleドライブのファイル・フォルダ操作をGASで自動化します |
| [Phase 3](./phase3.md) | スプレッドシート活用 | 3週 | スプレッドシートの操作とデータベース活用を学びます |
| [Phase 4](./phase4.md) | Gmailの送信 | 1週 | GASによるGmail送信とスプレッドシートを活用したメール配信システムを学びます |
| [Phase 5](./phase5.md) | Googleフォーム | 2週 | Googleフォームの作成・トリガー・DB操作を学びます |
| [Phase 6](./phase6.md) | Googleカレンダー | 2週 | Googleカレンダーのイベント操作とエラーハンドリングを学びます |
| [Phase 7](./phase7.md) | GASの便利な機能 | 1週 | プロパティストア・ダイアログ・ログ出力などGASの便利な機能を学びます |

## コンテンツ構成

各週には以下の3種類のコンテンツが含まれます。

| 種別 | 説明 |
|------|------|
| 動画 | YouTube解説動画 |
| スライド | PDFスライド資料（Supabase Storage） |
| 課題 | 実践演習（2〜3問）+ 模範回答 |
