# コンテンツ一覧

学習コンテンツの管理ディレクトリです。各講座ごとにサブディレクトリを設けています。

## 講座一覧

| 講座名 | ディレクトリ | フェーズ数 | 週数 |
|--------|-------------|-----------|------|
| [GAS学習](./gas/README.md) | `gas/` | 7 | 12 |

## ディレクトリ構成

```
contents/
├── README.md          # このファイル（講座一覧）
└── gas/               # GAS学習コース
    ├── README.md      # GAS講座概要・フェーズ一覧
    ├── phase1.md      # Phase 1 - 基礎文法
    ├── phase2.md      # Phase 2 - Googleドライブ
    ├── phase3.md      # Phase 3 - スプレッドシート活用
    ├── phase4.md      # Phase 4 - Gmailの送信
    ├── phase5.md      # Phase 5 - Googleフォーム
    ├── phase6.md      # Phase 6 - Googleカレンダー
    └── phase7.md      # Phase 7 - GASの便利な機能
```

## 新講座の追加方法

1. `contents/` 直下に講座名のディレクトリを作成する（例: `python/`）
2. `README.md`（講座概要）と各フェーズの `.md` ファイルを作成する
3. このファイルの講座一覧テーブルに追記する

## 各フェーズファイルの記載内容

各フェーズ `.md` ファイルには以下を記載します。

- フェーズ概要
- 各週の学習トピック
- 動画URL・スライドURL
- 課題（問題文 + 模範回答）
